## 🎨 主题特性分析

### 核心功能
基于 `komari-theme.json` 配置分析，该主题包含以下功能：

1. **外观设置**
   - 6种主题颜色（蓝、紫、绿、橙、粉、青）
   - 可调节玻璃透明度（0-100）
   - 可调节玻璃模糊度（0-100）
   - 可调节卡片圆角大小（8-40px）

2. **背景管理**
   - 桌面端背景图片设置
   - 移动端背景图片设置
   - 5种背景对齐方式

3. **布局控制**
   - 3种视图模式（网格、紧凑、表格）
   - 动画效果开关
   - 性能优化模式

4. **外观模式**
   - 跟随系统主题
   - 浅色模式
   - 暗黑模式

5. **全局统计**
   - 全局网速显示
   - 全局流量统计
   - 网速单位选择（MB/s 或 Mbps）

### 技术栈分析
基于 `dist/assets/` 文件分析：

- **React** - 主框架（vendor 包含 React 相关代码）
- **Framer Motion** - 动画库（motion-*.js）
- **Vite** - 构建工具（基于文件命名规则）
- **现代 ES 模块** - 使用 ES6+ 语法

## 📊 配置项详情

| 配置键 | 类型 | 默认值 | 范围/选项 |
|--------|------|--------|-----------|
| `theme_color` | select | 蓝色 | 蓝色,紫色,绿色,橙色,粉色,青色 |
| `glass_opacity` | number | 20 | 0-100 |
| `glass_blur` | number | 16 | 0-100 |
| `card_radius` | number | 24 | 8-40 |
| `bg_desktop_url` | string | - | URL 链接 |
| `bg_mobile_url` | string | - | URL 链接 |
| `bg_alignment` | select | 居中 | 居中,拉伸,平铺,顶部,底部 |
| `default_view_mode` | select | 网格 | 网格,紧凑,表格 |
| `enable_animations` | switch | true | true/false |
| `performance_mode` | switch | false | true/false |
| `appearance_mode` | select | 跟随系统 | 跟随系统,浅色模式,暗黑模式 |
| `show_global_speed` | switch | true | true/false |
| `show_global_traffic` | switch | true | true/false |
| `speed_unit` | select | MB/s | MB/s,Mbps |
