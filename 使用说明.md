# 🚀 Komari 液态玻璃主题使用说明

## 📦 一键构建

### Windows 用户
```cmd
双击 build.bat 文件
```

### Linux/Mac 用户
```bash
chmod +x build.sh
./build.sh
```

### 命令行方式
```bash
npm run build
# 或
node build.js
```

## 📋 构建要求

- **Node.js** - 任意版本（仅用于运行构建脚本）
- **项目文件完整** - 确保 `dist/` 目录和 `komari-theme.json` 存在

## 🎯 部署步骤

1. **运行构建脚本**
   - 生成 `komari-liquid-glass-theme.zip` 文件

2. **上传到 Komari**
   - 登录 Komari 管理面板
   - 进入主题管理页面
   - 上传 ZIP 文件

3. **启用主题**
   - 在主题列表中找到 "Liquid Glass iOS 26"
   - 点击启用

4. **配置主题**
   - 进入主题设置页面
   - 根据需要调整各项参数

## ✨ 主题特性

### 🎨 视觉效果
- **液态玻璃效果** - 真实的 iOS 26 风格
- **6种主题颜色** - 蓝色、紫色、绿色、橙色、粉色、青色
- **自定义透明度** - 0-100% 精确调节
- **动态模糊效果** - 0-100px 自由设置
- **圆角大小** - 8-40px 可调节

### 📱 移动端优化
- **响应式设计** - 完美适配各种屏幕
- **移动端专用背景** - 独立的移动端背景设置
- **触摸优化** - 专门的移动端交互

### 📊 功能丰富
- **全局统计** - 实时网速和流量总和
- **多视图模式** - 网格、紧凑、表格三种模式
- **智能主题** - 跟随系统、浅色、暗黑模式
- **性能优化** - 可关闭动画提升性能

## 🔧 故障排除

### 构建失败
1. 确保安装了 Node.js
2. 检查 `dist/` 目录是否存在
3. 确认 `komari-theme.json` 文件完整

### 主题无法加载
1. 确保 Komari 版本 >= 1.0.5
2. 检查主题包是否完整上传
3. 查看浏览器控制台错误

### 移动端显示问题
1. 设置移动端专用背景图片
2. 调整背景对齐方式
3. 检查网络连接

## 📞 技术支持

如果遇到问题：

2. 检查浏览器控制台错误信息
3. 确认服务器版本兼容性
